<h1>早速ツールをインストール</h1>
<hr>
早速やってみましょう。<br>
sudo apt install python3-pip<br>
sudo pip install youtube-dl<br>
でインストールできます。
<h1>使い方</h1>
<hr>
早速やってみましょう。<br>
ダウンロードしたいディレクトリに移動したら、youtube-dlコマンドにURLを引数として指定して実行してください。<br>
